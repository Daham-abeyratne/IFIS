# Income File Import System (IFIS)

## Overview
The **Income File Import System (IFIS)** is a JavaFX based desktop application designed to import, validate, display, and manage income records generated by the Income Recording Client System (IRCS). The system focuses on data integrity, error handling, and user interaction through a graphical interface.

IFIS reads structured income files, verifies record validity using checksums, and allows users to review, edit, and manage valid and invalid records before final processing.

---

## Key Features
- JavaFX based graphical user interface
- Import of income files generated by IRCS
- Automatic checksum validation
- Clear separation of valid and invalid records
- Editable table for income records
- Recalculation of checksums after updates
- Deletion of invalid or unwanted records
- Object oriented, modular design
